ps5.py is the main program file

To run the program, opening a command line terminal and type:
python ps5.py
 
The following files are necessary for the project to work and need to be included in the 
same directory as ps5.py.  Do not modify them.
feedparser.py
project_util.py
news_gui.py

ps5_test.py is a test suite to check our answers

triggers.txt is a configuration file that contains triggers, which are rules that generate alerts
